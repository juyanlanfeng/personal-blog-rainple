OC 展示页（/oc/）媒体管理说明
==================================

你只需要管理下面四个文件夹里的图片/视频，页面自动读取，不用改代码。
（dev 下增删后刷新页面即可；正式部署需重新 npm run build。）

四个文件夹（都在 public\oc\ 下）：

  art\featured\       画作 · 精选，显示在 /oc/ 主页
  art\more\           画作 · 其余，收进相册整页
  fursuit\featured\   毛照 · 精选，显示在 /oc/ 主页
  fursuit\more\       毛照 · 其余，收进相册整页

------------------------------------------------------------
主页 vs 相册整页
------------------------------------------------------------
- /oc/ 主页：每个区块只展示 featured 文件夹里的【图片】（精选门面）。
- 点区块下方「查看全部 N 项 →」按钮，进入相册整页：
    画作 → /oc/art/      毛照 → /oc/fursuit/
  相册整页展示该分类的【全部】图片和视频（featured + more 一起）。

------------------------------------------------------------
支持的格式
------------------------------------------------------------
- 图片：jpg / jpeg / png / webp / gif / avif
- 视频：mp4 / webm / mov / m4v
  视频在相册里显示首帧 + 播放按钮，点开在大图里播放。

------------------------------------------------------------
标题（可选）
------------------------------------------------------------
- 默认：普通文件名（IMG_xxxx、数字等）不显示标题，只显示画面，干净。
- 想要标题：把文件名里写上【中文】，格式  [序号_]中文_English.ext
    例：  01_竹林温泉_Bamboo hot spring.jpg
    - 开头序号（01、02…）只用于排序，不显示
    - 第一个下划线前是中文标题，后面是英文小字（可省略）

------------------------------------------------------------
排序
------------------------------------------------------------
- 按文件名排序。想控制顺序就在文件名前加 01_、02_… 这样的序号。
